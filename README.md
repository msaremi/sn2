# Memory-Efficient SEMNAN Solver CUDA Torch Extension

## Installation

Installation requires the NVCC compiler with an ABI-compatible C++ compiler. 
A pre-compiled egg is available [here](dist).

To intall, use:
```shell
cd u-semnan
python setup.py install
```
